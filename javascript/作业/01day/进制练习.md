    二进制（逢二进一）

    十六进制（逢十六进一）

二进制变为十进制
    二级制：1101
    十进制：1*2^3+1*2^2+0*2^1+1*2^0 = 8 + 2 + 1 =11

十进制变为二进制
    十进制：15
    二级制：
        15/2 = 7...1
        7/2  = 3...1
        3/2  = 1...1
        1/2  = 0...1
        从下到上算二进制为：1111

其它进制与十进制的换算跟二级制与十进制的换算一样
